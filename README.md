# otDB
## 개요
인터넷db 기말프로젝트<br>
오라클 데이터베이스의 샘플 데이터를 ORM으로 변환 하여 가상으로 존재하는 컴퓨터 부품 판매점의 재고 관리를 웹으로 구현한 것 입니다. <br>
<br>
이 저장소는 기존 저장소를 private 처리 후 다시 생성된 저장소 입니다. <br>

## 기술스택
* OpenJDK 11
* SpringBoot 2.4.5
* MariaDB
* Linux(ubuntu, Oracle Cloud)
* Docker
* Thymeleaf
